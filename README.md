# Mini CRM

A simple CRM application built with Next.js and Tailwind CSS.

## Features
- **Deals Management**: Track deals with stages, amounts, and owners.
- **Invoices**: Manage billing and payments.
- **Dashboard**: Overview of recent activity and alerts.

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- Shadcn UI (Customized as Yohaku)
